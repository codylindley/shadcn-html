shadcn-html / dropdown

Minimal JS

Dropdown Menu

Context menu built on the popover API. Light-dismiss is automatic. Supports groups, shortcuts, separators, and destructive items. Animated with @starting-style.

patterns/dropdown.md

Native basis

popover API + <button> trigger. Light-dismiss and top-layer rendering are automatic.

Item types

role="menuitem"Standard item — click activates, menu closes role="menuitemcheckbox"Toggleable item with .dropdown-check class role="menuitemradio"Exclusive choice with .dropdown-radio class disabledGreyed out, not focusable via keyboard

Wiring conventions

• data-dropdown-trigger="[id]" on button → opens that menu

• role="menu" with popover attribute on menu container

• tabindex="-1" on all items — JS manages highlight via data-highlighted

Keyboard

ArrowDown/Up cycle items. Enter/Space activate or toggle checkbox/radio. Escape closes.

Typeahead: pressing a letter focuses the first matching item.

<!-- Trigger -->
<button class="btn" data-variant="outline"
        aria-haspopup="menu" aria-expanded="false"
        aria-controls="my-menu"
        data-dropdown-trigger="my-menu">
  Options
  <svg aria-hidden="true">...</svg>
</button>

<!-- Menu -->
<div id="my-menu" role="menu" popover
     class="dropdown-content"
     aria-label="Options menu">
  <div class="dropdown-label">Section</div>
  <button role="menuitem" class="dropdown-item"
          tabindex="-1">
    Profile
    <span class="dropdown-shortcut">⇧⌘P</span>
  </button>
  <button role="menuitem" class="dropdown-item"
          tabindex="-1" disabled>Upload</button>
  <div class="dropdown-separator" role="separator"></div>
  <button role="menuitem" class="dropdown-item"
          data-variant="destructive" tabindex="-1">
    Delete
  </button>
</div>

<!-- Checkbox items -->
<button role="menuitemcheckbox"
        class="dropdown-item dropdown-check"
        aria-checked="true" tabindex="-1">
  Status Bar
</button>

<!-- Radio items (wrap in role="group") -->
<div role="group" aria-label="Layout">
  <button role="menuitemradio"
          class="dropdown-item dropdown-radio"
          aria-checked="true" tabindex="-1">
    Sidebar Left
  </button>
  <button role="menuitemradio"
          class="dropdown-item dropdown-radio"
          aria-checked="false" tabindex="-1">
    Sidebar Right
  </button>
</div>
