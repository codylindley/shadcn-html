shadcn-html / introduction

The AI prototyping
substrate.

Semantic HTML. Shadcn-compatible tokens. Tailwind v4 static bridge. No framework. No runtime. No npm install. Just the lowest-level web primitives assembled into a system any AI can understand and build from.

AI Optimized

Pattern specs, not components. AI reads and builds from scratch every time.

Framework Free

Runs in any browser, zero dependencies, no build pipeline required.

The contract.

Three layers. One CSS file each. Everything else follows.

the full system in ~30 lines 
/* 1 — tokens.css: define the design system */
:root {
  --background:         oklch(1 0 0);
  --foreground:         oklch(0.145 0.005 285);
  --primary:            oklch(0.205 0.005 285);
  --primary-foreground: oklch(0.985 0.002 247);
  --muted:              oklch(0.94 0.003 247);
  --muted-foreground:   oklch(0.52 0.01 285);
  --border:             oklch(0.88 0.004 247);
  --ring:               oklch(0.205 0.005 285);
  --radius:             0.625rem;
  --font-sans:          ui-sans-serif, system-ui, sans-serif;
  /* ...sidebar, chart, shadow, spacing tokens */
}
.dark { /* same keys, dark values */ }

/* 2 — theme.css: bridge to Tailwind v4 */
@custom-variant dark (&:is(.dark *));
@theme inline {
  --color-primary:            var(--primary);
  --color-primary-foreground: var(--primary-foreground);
  --color-border:             var(--border);
  --font-sans:                var(--font-sans);
  --radius-sm:                calc(var(--radius) - 4px);
  --radius-md:                calc(var(--radius) - 2px);
  --radius-lg:                var(--radius);
  --radius-xl:                calc(var(--radius) + 4px);
  /* ...all color + shadow mappings */
}
@layer base {
  * { @apply border-border outline-ring/50; }
  body { @apply bg-background text-foreground; }
}

/* 3 — use in HTML */
<button class="btn" data-variant="default">Save</button>
<div class="card">...</div>
