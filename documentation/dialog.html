shadcn-html / dialog

Minimal JS

Dialog

Modal window built on native <dialog> + showModal(). Focus trap, Escape key, and backdrop are all browser-native. Animated with @starting-style.

patterns/dialog.md

Native basis

<dialog> + .showModal(). Browser provides focus trap, Escape key, and ::backdrop.

Wiring conventions

• data-dialog-trigger="[id]" on any element → opens that dialog

• data-dialog-close on any element inside → closes the dialog

• Click on backdrop → closes (click lands on <dialog> itself)

ARIA

• role="dialog" + aria-modal="true" + aria-labelledby

• Focus returns to trigger on close (JS)

Animation

CSS-only enter via @starting-style. Exit via transition + allow-discrete.

<!-- Trigger -->
<button class="btn" data-variant="default"
        data-dialog-trigger="my-dialog"
        aria-haspopup="dialog">
  Open
</button>

<!-- Dialog (place as direct child of body) -->
<dialog id="my-dialog" class="dialog"
        role="dialog" aria-modal="true"
        aria-labelledby="my-dialog-title">
  <div class="dialog-content">
    <div class="dialog-header">
      <h2 class="dialog-title" id="my-dialog-title">Title</h2>
      <p class="dialog-description">Description text.</p>
    </div>

    <!-- body content -->

    <div class="dialog-footer">
      <button class="btn" data-variant="outline"
              data-dialog-close>Cancel</button>
      <button class="btn" data-variant="default">
        Save
      </button>
    </div>
  </div>
</dialog>

<!-- dialog.js: wire once for all dialogs on page -->
<script>
document.querySelectorAll('[data-dialog-trigger]').forEach(trigger => {
  const dialog = document.getElementById(trigger.dataset.dialogTrigger);
  if (!dialog) return;
  trigger.addEventListener('click', () => {
    dialog._trigger = trigger;
    dialog.showModal();
  });
});

document.querySelectorAll('dialog').forEach(dialog => {
  dialog.addEventListener('click', e => {
    if (e.target === dialog) dialog.close();
  });
  dialog.querySelectorAll('[data-dialog-close]').forEach(btn => {
    btn.addEventListener('click', () => dialog.close());
  });
  dialog.addEventListener('close', () => {
    dialog._trigger?.focus();
  });
});
</script>
