Tabs

Organize content into switchable panels. Uses WAI-ARIA Tabs pattern with roving tabindex and arrow key navigation. Available in pill and line variants.

patterns/tabs.md

Native basis

role="tablist" + role="tab" + role="tabpanel". No native element — requires JS for keyboard nav and panel switching.

Variants

pill (default)--muted bg, active tab has --background bg + shadow data-variant="line"transparent bg, --border bottom, active: --primary underline

Orientation

horizontal (default)ArrowRight/Left navigate. Tabs above panels. aria-orientation="vertical"ArrowDown/Up navigate. Tabs beside panels.

Disabled state

Add disabled attribute to tab button. Keyboard navigation skips disabled tabs automatically.

ARIA

Roving tabindex: active tab = tabindex="0", others = -1.

Panels use aria-labelledby, inactive panels get hidden.

Keyboard

ArrowRight/Left cycle tabs. Home/End jump to first/last. Tab moves into the active panel.

<!-- Pill variant (default) -->
<div class="tabs">
  <div class="tab-list" role="tablist" aria-label="Settings">
    <button class="tab-trigger" role="tab"
            aria-selected="true"
            aria-controls="panel-account"
            id="tab-account">Account</button>
    <button class="tab-trigger" role="tab"
            aria-selected="false"
            aria-controls="panel-password"
            id="tab-password"
            tabindex="-1">Password</button>
    <button class="tab-trigger" role="tab"
            aria-selected="false" disabled
            tabindex="-1">Billing</button>
  </div>

  <div class="tab-content" role="tabpanel"
       id="panel-account"
       aria-labelledby="tab-account"
       tabindex="0">
    Account content...
  </div>
  <div class="tab-content" role="tabpanel"
       id="panel-password"
       aria-labelledby="tab-password"
       tabindex="0" hidden>
    Password content...
  </div>
</div>

<!-- Line variant -->
<div class="tab-list" role="tablist" data-variant="line">
  ...
</div>

  </div>
